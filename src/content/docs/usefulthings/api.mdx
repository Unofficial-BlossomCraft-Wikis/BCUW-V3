## Internal API
these API's where made for internal use but are available to the public.

### Contributors Image API
:::caution
This updates daily at midnight UTC. It is not updated at time of the request.
:::

The contributors image API is how the [staff](/contributing/staff) page get it's images.
syntax:

```html title="example.html"
<img src="https://www.bcuw.xyz/api/v1/imgs/contributor/{github username}.png" />
<img src="https://www.bcuw.xyz/api/v1/imgs/contributor/{github username}.svg" />
```

#### example code:
```html title="example.html"
<img src="https://www.bcuw.xyz/api/v1/imgs/contributor/altie122.png" />
<img src="https://www.bcuw.xyz/api/v1/imgs/contributor/altie122.svg" />
```
##### example output:
![altie122](https://www.bcuw.xyz/api/v1/imgs/contributor/altie122.png)

This updates daily at midnight UTC. It is not updated at time of the request.
:::
