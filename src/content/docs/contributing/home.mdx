## In what ways can you contribute?

You can contribute to the BCUW in many ways. Here are a few of the many ways you can contribute:

- [Report a bug](#report-a-bug)
- [Add a resource](#add-a-resource)
- [Add assets](#add-assets)
- [Translate](#translate)
- [Add crates or items](#add-crates-or-items)

for questions or for a full and more detailed list feel free to DM @altie122 on discord.

## Report a bug

If you find a bug in the BCUW, please [open an issue](https://github.com/Unofficial-BlossomCraft-Wikis/BCUW/issues) and let us know what you think is wrong. We'll do our best to fix it as soon as possible.

## Add a resource

If you have a resource you'd like to add to the BCUW, please submit it [here](https://forms.gle/Qx2eANJwPzGGLbZN6).

## Add assets

If you would like to help with assets (3D models, item lore, etc), please DM @altie122 on discord for more information.

## Translate

We would like to translate BCUW. If you would like to help with translations, please DM @altie122 on discord for more information.
